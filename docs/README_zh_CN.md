# 文档目录（中文）

## 通用与发布
- `CHANGELOG.md`：版本更新日志。
- `ISSUES_STATUS.md`：当前问题追踪状态。
- `upstream_feature_backlog.md`：上游功能待办与回移项。
- `RELATED_PROJECTS.md` / `RELATED_PROJECTS_RESEARCH.md`：相关项目与调研。

- `INPAINTER_VRAM_OPTIONS.md` / `INPAINTING_QUALITY_AND_SPEED.md`：修复模型显存与质量速度权衡。
